# car-maintenance-app
Araç Bakım Takip Uygulaması  Bu proje, React, Vite ve Tailwind CSS kullanılarak geliştirilmiş modern bir araç bakım takip uygulamasıdır. Uygulama, kullanıcıların araçlarına ait bakım işlemlerini kolayca kaydetmesini, düzenlemesini ve takip etmesini amaçlamaktadır.  Uygulama tam CRUD desteği sunar. Kullanıcılar yeni bakım kayıtları ekleyebilir, mevcut kayıtları görüntüleyebilir, düzenleyebilir ve silebilir. Bakım kayıtları türlerine göre filtrelenebilir ve tarih sırasına göre listelenir. Toplam bakım sayısı, toplam maliyet ve son kilometre bilgileri istatistik kartları üzerinden görüntülenir.  Tüm veriler tarayıcının localStorage yapısında saklanır ve sayfa yenilense bile kaybolmaz. Uygulama responsive bir tasarıma sahiptir ve mobil, tablet ve masaüstü cihazlarda sorunsuz çalışır.  Proje React framework’ü ile geliştirilmiş olup Vite build aracı kullanılmıştır. Stil yönetimi için Tailwind CSS tercih edilmiştir. Veri yönetimi React Hooks (useState, useEffect) ile sağlanmıştır. Proje yapısı modülerdir ve Components, Pages ve Interfaces klasörleri altında organize edilmiştir.  Proje Netlify üzerinde yayınlanmıştır ve canlı sürüme aşağıdaki bağlantıdan erişilebilir. https://car-maintenance-app-talip.netlify.app/  Bu proje, web geliştirme eğitimi kapsamında modern frontend teknolojilerini göstermek amacıyla hazırlanmıştır.
